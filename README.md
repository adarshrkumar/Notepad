# Notepad

A web-based text and code editor with cloud storage and rich text editing capabilities.

## Features

- **Rich Text Editor**: Powered by TinyMCE with full formatting support
- **Code Editor**: Syntax-highlighted code editing mode
- **File Management**:
  - Create new files
  - Open existing files from cloud storage
  - Upload files from your computer
  - Download files
  - Share files via link
- **Image Support**: Preview and store images
- **Auto-save**: Content saves automatically on changes
- **Markdown Export**: Converts content to Markdown using TurnDown

## Pages

- `/` - Home page with file actions
- `/editor.html` - Rich text editor (TinyMCE)
- `/code-editor.html` - Code editor with syntax highlighting
- `/open.html` - Browse and open existing files
- `/preview.html` - Preview files
- `/settings.html` - Editor settings

## Tech Stack

- HTML/CSS/JavaScript
- TinyMCE for rich text editing
- TurnDown for HTML-to-Markdown conversion
- Material Symbols for icons
- Hosted via GitHub Pages
